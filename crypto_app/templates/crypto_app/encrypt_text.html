Comment utiliser le chiffrement de texte
Pour chiffrer
  1. Saisissez votre texte confidentiel
  2. Choisissez un mot de passe fort
  3. Sélectionnez "Chiffrer"
  4. Copiez le résultat chiffré
Pour déchiffrer
  1. Collez le texte chiffré
  2. Saisissez le même mot de passe
  3. Sélectionnez "Déchiffrer"
  4. Récupérez le texte original
Conseils pratiques
  • Utilisez un mot de passe différent pour chaque message
  • Partagez le mot de passe par un canal séparé (pas dans le même message)
  • Vérifiez le format : le texte chiffré est au format JSON
  • Sauvegardez vos mots de passe dans un gestionnaire sécurisé
